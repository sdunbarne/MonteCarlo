# MonteCarlo
Section 10 of tentative book on Markov chains, with motivation for the Metropolis and Metropolis-Hastings algorithms for Markov Chain Monte Carlo, along with some theory. 

The heart of the MCMC method is the inverse problem:  Given a
distribution \( \pi \) on a finite state space, find an
irreducible, aperiodic Markov chain that is stationary on \( \pi\).
The solution to this inverse problem is the Metropolis algorithm and the 
Metropolis-Hastings Algorithm.
